# Процедура анализа здоровья HealthVault

## Когда применять

Пользователь просит проанализировать:
- тело, вес, состав тела
- питание, КБЖУ
- сон
- тренировки, активность

## Обязательный шаг 1: Запуск валидатора

Перед анализом **всегда** запускай:

```bash
cd /Users/alexlyskovsky/HealthVault
python3 scripts/validate_health_data.py
```

## Шаг 2: Интерпретация результата

- **СТАТУС: КРИТИЧНО** — не проводить полный анализ. Сообщить пользователю о проблемах с данными.
- **СТАТУС: ПРЕДУПРЕЖДЕНИЕ** — анализ можно проводить, но обязательно указать ограничения из отчёта.
- **СТАТУС: ХОРОШО** — данные готовы к анализу.

## Шаг 3: Провести анализ

Используй данные из:
- `data/nutrition/nutrition_log.json` — питание
- `data/apple-health/` — Apple Health (вес, состав тела, активность)
- `data/health_dashboard.json` — сводная статистика
- `data/weight_trends_analysis.json` — тренды веса

## Важно

Не делай выводов на основе неполных данных. Если валидатор показал неполные дни — исключи их из анализа и укажи это в отчёте.
