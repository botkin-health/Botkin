# Импорт данных Apple Health

Этот workflow описывает процесс еженедельного импорта данных из Apple Health в базу HealthVault.

## Когда использовать
Запускать **каждую неделю** (например, по понедельникам) для обновления данных о весе, давлении и пульсе.

## Что импортируется
- **Вес** - все замеры + средний вес по дням
- **Артериальное давление** - систолическое/диастолическое
- **Пульс в покое** - данные от Apple Watch/iPhone

## Инструкция

### 1. Экспорт данных из Apple Health
На iPhone:
1. Открыть приложение **Здоровье** (Health)
2. Нажать на иконку профиля (правый верхний угол)
3. Прокрутить вниз → **Экспортировать данные о здоровье**
4. Дождаться подготовки экспорта (может занять несколько минут)
5. Выбрать **Сохранить в файлы** → **iCloud Drive** → **Загрузки**

### 2. Дождаться синхронизации с Mac
Файл `export.zip` должен появиться в папке `/Users/alexlyskovsky/Downloads`

### 3. Запустить импорт

// turbo
```bash
cd /Users/alexlyskovsky/HealthVault
python3 scripts/import_apple_health.py
```

Скрипт автоматически:
- Найдет последний экспорт в Downloads
- Распакует его
- Извлечет данные о весе, давлении и пульсе
- Сохранит в базу HealthVault:
  - `data/apple_health_weight.json` - все замеры веса
  - `data/apple_health_weight_daily.json` - средний вес по дням
  - `data/apple_health_blood_pressure.json` - данные давления
  - `data/apple_health_heart_rate.json` - пульс в покое
- Удалит временные файлы

### 4. Проверить результаты

// turbo
```bash
# Последние 5 дней веса
tail -15 data/apple_health_weight_daily.json

# Последние замеры давления
tail -15 data/apple_health_blood_pressure.json
```

## Что делать дальше

После импорта данные о весе и давлении можно использовать для:
- Анализа динамики веса
- Отслеживания артериального давления
- Корреляции с питанием и тренировками
- Построения графиков и трендов

## Полезные команды

### Посмотреть статистику импорта
```bash
wc -l data/apple_health_weight.json
wc -l data/apple_health_blood_pressure.json
```

### Найти последний вес
```bash
grep -A 3 "date.*$(date +%Y-%m-%d)" data/apple_health_weight_daily.json
```

### Найти последнее давление
```bash
tail -10 data/apple_health_blood_pressure.json | grep -E "(date|systolic|diastolic)"
```

## Troubleshooting

**Проблема**: Файл export.zip не найден  
**Решение**: Убедитесь, что экспорт завершился на iPhone и синхронизировался с iCloud

**Проблема**: Скрипт падает с ошибкой памяти  
**Решение**: Закрыть другие приложения, скрипт использует потоковый парсинг для экономии памяти

**Проблема**: Данные не обновляются  
**Решение**: Проверьте, что экспорт содержит свежие данные (смотрите дату последней записи в выводе скрипта)
