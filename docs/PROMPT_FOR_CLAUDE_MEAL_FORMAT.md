# Инструкция для Claude: формат питания под HealthVault

**Когда обсуждаешь со мной обед/ужин и даёшь таблицу КБЖУ — перепиши её в таком формате (чистый текст, без markdown-таблицы), чтобы я мог скопировать и вставить в Telegram-бот HealthVault:**

```
[Приём пищи]: 
[продукт 1] [вес]г — [калории] ккал (Б[белки] Ж[жиры] У[углеводы])
[продукт 2] [вес]г — [калории] ккал (Б[белки] Ж[жиры] У[углеводы])
...
```

**Правила:**
- Первая строка: `Обед:` или `Ужин:` и т.п. (название приёма).
- Каждый продукт — отдельная строка.
- Вес в граммах с суффиксом `г` (103г, 131.5г).
- Калории и макросы в скобках: `(Б6 Ж16 У25)` — белки, жиры, углеводы в граммах.
- Без emoji, без таблиц, без markdown. Просто текст.

**Пример для моего обеда с блинами:**
```
Обед:
блин с капустой и яйцом 103г — 263 ккал (Б6 Ж16 У25)
блин Морской Богатырь 131.5г — 363 ккал (Б14.5 Ж23.5 У24)
блин с красной икрой 80.5г — 246 ккал (Б11 Ж13 У22)
ватрушка 108г — 318 ккал (Б5 Ж20 У29)
```

Я скопирую этот текст и отправлю боту — он распознает продукты, сохранит в базу и покажет кнопку «Сохранить».
