# Дизайн: Редизайн экрана дневника питания (Telegram Mini App)

**Дата:** 2026-04-19
**Статус:** Approved — готово к реализации
**Область:** `telegram-bot/webapp/` (day.js, day.css, index.html)

---

## Цель

Переработать главный экран дневника питания в Telegram Mini App так, чтобы:
1. Все 4 слота (завтрак/обед/перекус/ужин) в свёрнутом виде помещались без скролла на iPhone (~700px рабочей высоты).
2. Информационная иерархия была чёткой: главное число сразу бросается в глаза.
3. Интерфейс читался комфортно одной рукой.
4. Была точка входа в другие разделы (добавки, настройки) через таб-бар внизу.

---

## Принятые решения (итог брейнсторма)

| Вопрос | Выбор |
|---|---|
| Направление | Информационная иерархия (акцент на самом важном числе) |
| Герой-элемент | Крупный цифровой баннер (−603 ккал / +324 ккал) |
| Стиль слотов | Цветная левая полоса, цвет ккал = цвет полосы |
| Типографика | Читаемый масштаб (+2px везде от текущего) |
| Заголовок | Оптимизированный: одна строка вместо двух |
| Настройки | Таб-бар снизу (Дневник · Добавки · Настройки) |

## Компонент 1 — Заголовок (Header)

**Было:** две строки (~76px):
- строка 1: `🍽 Дневник` + `⚙️`
- строка 2: `‹` + `Пт, 17 апр.` + `›`

**Станет:** одна строка (~40px):
```
‹     ДНЕВНИК      ›
      Пт, 17 апр.
```
- Левый элемент: `‹` (22px, #2a7ae2)
- Центр: подзаголовок `ДНЕВНИК` (13px, uppercase, muted) + дата `Пт, 17 апр.` (15px, bold)
- Правый элемент: `›` (22px, #2a7ae2)
- Padding: 8px 12px
- Экономия: ~36px по высоте

---

## Компонент 2 — Баннер (Hero)

Главный элемент экрана. Показывает баланс калорий за день.

**Формат при перерасходе:**
```
−603 ккал          ← 34px, bold 900, #ff3b30
перебор · съедено 2465 из 1862  ← 12px, #636366
Б 145/140  Ж 145/62  У 145/186  ← 12px, цветные
```

**Формат при остатке:**
```
+324 ккал          ← 34px, bold 900, #34c759
норма · съедено 1538 из 1862
Б 98/140  Ж 40/62  У 110/186
```

**Цветовая логика макросов:**
- `.ok` (85–110% нормы): `#34c759`
- `.warn` (< 85%): `#ff9500`
- `.over` (> 110%): `#ff3b30`
- цель `/NNN`: `#8e8e93`, font-weight 400

Padding баннера: 6px 14px 8px. Border-bottom: 1px solid #e5e5ea.

---

## Компонент 3 — Слоты приёмов пищи

Четыре слота: Завтрак, Обед, Перекус, Ужин.

**Цветовая схема слотов:**

| Слот | Цвет полосы | Цвет ккал |
|---|---|---|
| 🌅 Завтрак | #f5b02b (жёлтый) | #f5b02b |
| ☀️ Обед | #4cb563 (зелёный) | #4cb563 |
| 🥗 Перекус | #e0577c (розовый) | #e0577c |
| 🌙 Ужин | #8e79d6 (фиолетовый) | #8e79d6 |

**Разметка одного слота:**
```
[4px цветная полоса] [🌅 Завтрак        ] [1 184]
                     [Тунец, яйца, кап…]
```
- Карточка: белая, border-radius 10px, border-left 4px solid {цвет}
- Padding: 9px 12px
- Название: 13px, font-weight 700
- Превью: 11px, #8e8e93, truncated (max-width 160px)
- Ккал: 13px, font-weight 700, цвет слота
- Пустой слот: превью и ккал — `#c7c7cc`, ккал = `—`
- Gap между слотами: 6px
- Фон секции: #f2f2f7, padding 8px

---

## Компонент 4 — Прогресс-панель (Footer)

Показывает только клетчатку и белки (ккал уже есть в баннере, жиры/углеводы — в макросах).

```
ПРОГРЕСС
Клетчатка ████████░░  26/30г
Белки     ██████████  145/140г
```

- Заголовок: 9px, uppercase, #8e8e93, letter-spacing .5px
- Бар-строка: grid 56px 1fr 64px, gap 6px, font-size 11px
- Трек: #e5e5ea, высота 5px, border-radius 3px
- Заполнение клетчатки: #8e79d6
- Заполнение белков: #e0577c
- Цвет значений: `.ok`/`.warn`/`.over` по той же логике
- Padding: 8px 14px 10px

---

## Компонент 5 — Таб-бар (Bottom Navigation)

Новый компонент. Постоянно виден внизу экрана.

```
[📅 Дневник]  [💊 Добавки]  [⚙️ Настройки]
```

- Высота: ~50px (с учётом safe area на iPhone)
- **Палитра: Warm Stone** — тёплый бежево-серый, отделяет навигацию от белого контента
  - Фон: `#e8e3dc`
  - Border-top: `1px solid #d6d0c9`
  - Активная подпись/иконка: `#5a5048` (тёмно-коричневый)
  - Неактивная подпись: `#9a9187` (приглушённый)
- Размер иконок: 22px
- Подпись: 10px, font-weight 600

**Вкладки v1:**
- `📅 Дневник` — текущий экран (day.js)
- `💊 Добавки` — страница добавок (supplements, заглушка в v1)
- `⚙️ Настройки` — цели по нутриентам, параметры (заглушка в v1)
