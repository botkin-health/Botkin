## Как Нике получить BMR и активные из Apple Health

1. iPhone → Здоровье → Обзор → Энергия.
2. **Базовая энергия** — суточный BMR (среднее за неделю/месяц).
3. **Активная энергия** — сжигание от движения (среднее).
4. TDEE ≈ BMR + Активная.  
   Пример: BMR 1400 + Активная 250 = 1650 ккал.

Ввести в бота: `/setup BMR 1400, активные 250`.
