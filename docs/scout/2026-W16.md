## #code — репозитории и рабочие инструменты (4)

### 1. [Pace — MCP Server для wearables](https://news.ycombinator.com/item?id=47610713)  `#code` 🗳️20 очков HN

**Что:** Claude Connector, который подключает 20+ wearables (Garmin, Polar, Whoop) напрямую к Claude через MCP-протокол. Задаёшь вопрос «как я спал» — Claude дёргает твои данные сам.

**Почему тебе:** Это прямой альтернативный путь к тому, что ты строишь. Вместо «Garmin → Python → PostgreSQL → LLM-скрипт» — «Garmin → MCP → Claude». Стоит посмотреть что они забирают из API Garmin и как форматируют для LLM — можно украсть паттерны для твоего пайплайна.

**Что попробовать:** открыть дискуссию на HN, посмотреть есть ли open source (скорее всего закрытый SaaS, но автор может делиться архитектурой). Если закрыт — читать только HN-тред ради идей.

### 4. [RuView — vital signs из WiFi-сигналов](https://github.com/ruvnet/RuView)  `#code` ⭐46 922 звёзд

**Что:** Превращает обычный WiFi-роутер в датчик pose estimation и **мониторинг витальных показателей без wearables**. WiFi DensePose на коммодити-железе.

**Почему тебе:** Экзотика, но концептуально очень релевантная — представь мониторинг сна Валерии (72 года, Новосибирск) без того чтобы заставлять её носить Garmin. Посмотреть что реально работает — и стоит ли закупать роутер в её квартиру.

## #research — исследования (3)

### 5. [Effects of omega-3 fatty acid supplementation on heart rate variability](https://pubmed.ncbi.nlm.nih.gov/40085421/)  `#research` 📅 2026-03-09

**Что:** Свежая публикация в World J Clin Pediatr, метаанализ омега-3 добавок и HRV.

**Почему тебе:** В `LONGEVITY_BENCHMARKS.md` омега-3 индекс стоит как 🔴 высокий приоритет. Это исследование даёт тебе конкретный biomarker-эффект (HRV) который ты УЖЕ меришь через Garmin — значит можешь проверить эффект добавок на себе с N=1 экспериментом.

**Что попробовать:** сдать омега-3 индекс, начать добавку, мерить недельный HRV через Garmin до/после 8 недель. Записать в `knowledge_base.json`.

### 6. [Effects of exercise interventions on 24-h continuous glucose profiles in non-diabetic populations](https://pubmed.ncbi.nlm.nih.gov/40071890/)  `#research` 📅 2026-03, Diabetes Obes Metab

**Что:** Как упражнения влияют на 24-часовой профиль глюкозы у **здоровых людей** (не диабетиков).

**Почему тебе:** Главный аргумент «зачем CGM здоровому человеку». В профиле ты поставил CGM в high_priority — это исследование поможет выбрать протокол (когда тренироваться относительно еды, как мерить ответ).

### 7. [Wearable AI in the Era of Large Sensor Models](https://arxiv.org/abs/2604.08117)  `#research` 📅 arXiv 2026-04-11

**Что:** Концептуальный обзор того, куда движутся wearable AI — от task-specific моделей (один сенсор = один алгоритм) к большим sensor foundation models (один универсальный «GPT для телесных данных»).

**Почему тебе:** Долгосрочная рамка для HealthVault. Сейчас у тебя LLM интерпретирует **результаты** (анализы, знания), но не **сырые сенсорные потоки**. Этот обзор — карта куда можно двигаться после базового дашборда.
