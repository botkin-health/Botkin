
Schema для секции medications в knowledge_base.json
=====================================================

Структура для Rx-препаратов (НЕ БАДов — те идут в user_settings.supplements):

  "medications": [
    {
      "name": "Финастерид",
      "active_ingredient": "Финастерид",
      "dose": "5 мг",
      "schedule": "1 раз в день утром",
      "purpose": "Гиперплазия простаты / повышенный ПСА",
      "prescribed_by": "уролог Иванов И.И.",
      "started": "2024-11-15",
      "ended": "2025-02-15",          # null если ещё принимает
      "status": "completed",           # active / completed / paused / cancelled
      "notes": "Курс 3 месяца, контрольный ПСА не сдавал"
    }
  ]

Используется агентом для отвечать на 'какие препараты я сейчас принимаю',
'когда заканчивается курс', 'когда последний раз пил X'.
БАДы и витамины НЕ сюда — для них есть user_settings.supplements (постоянный
режим) + supplements_log (фактические приёмы в боте).
