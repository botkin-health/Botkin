# 📊 Dashboard — твоя большая панель здоровья

> Если Mini-app — это «что я ел сегодня», то Dashboard — это «как я живу последние полгода». Полная картина: вес и его динамика, анализы крови с трендами, тренировки, давление, сон, графики биомаркеров, прогноз биологического возраста.

## Как открыть

У каждого юзера есть **личная ссылка вида:**

```
https://health.orangegate.cc/mc/{share_token}
```

`share_token` — уникальный длинный токен. Открывается без пароля (security-by-obscurity): если знаешь токен — видишь дашборд.

### Где взять свой share-link

- В мини-апп → Настройки → «Открыть дашборд»
- В Telegram-боте: команда `/dashboard` (если настроена)
- В welcome-сообщении после онбординга

### Можно ли расшарить друзьям

Да. **Это безопасно настолько, насколько ты решишь**: если поделишься ссылкой — человек увидит твой дашборд. Не делись если не хочешь.

Можно **пересоздать share_token** в настройках мини-аппа — старая ссылка перестанет работать.

## Что внутри дашборда

Дашборд **адаптивный** — показывает только те блоки, для которых есть данные. У нового юзера будет 2-3 блока, у активного — 15-20.

### Шапка — ключевые KPI

- Возраст / биологический возраст (если есть PhenoAge)
- Текущий вес и тренд (↘ −2.3 кг за месяц)
- Главный риск-маркер (например, ApoB или ферритин если выбивается)
- SCORE2 или ASCVD-риск ССЗ если есть данные

### Блок «Антропометрия»

- Вес, % жира, мышечная масса — графики за весь период замеров
- BMI
- Объёмы (талия, шея, бицепс) если ведёшь
- Тренд: положительная/отрицательная динамика к цели

### Блок «Гемодинамика»

- Артериальное давление — график систолическое/диастолическое
- Пульс покоя — линия с разбивкой день/ночь
- HRV (вариабельность пульса) с Garmin/Apple Watch

### Блок «Активность»

- Шаги по дням
- Активные ккал
- Тренировки (Zone 2, силовые, HIIT) с разбивкой по типам
- Цель vs факт за период

### Блок «Сон»

- Часы сна по ночам
- Структура (deep / REM / light)
- Эффективность

### Блок «Питание»

- Дневные калории за период с целью
- Распределение БЖУ
- Кпд: сколько целевых дней / профицитных
- Топ-съеденных продуктов

### Блок «Анализы крови»

- Все биомаркеры из `knowledge_base.json` с динамикой
- Цветовая кодировка: 🟢 норма · 🟡 пограница · 🔴 выбивается
- Графики за все имеющиеся годы
- Сравнение с целевыми (твоими или популяционными для возраста/пола)

### Блок «Лекарства и добавки»

- Adherence-график: процент дней приёма за период
- Список текущих с дозами
- История смен схемы (особенно для тех у кого хроническая терапия)

### Блок «Risk / Longevity»

- PhenoAge / GrimAge (если посчитано)
- SCORE2 / ASCVD (если посчитано)
- LE8 score
- Сводка по факторам риска и зонам внимания

## Откуда данные

Дашборд **читает только сервер-БД**. Не локальные файлы юзера, не приватные дневники. Точно то, что собирается из:
- Twoего Telegram-бота (питание, добавки, вес, АД)
- Apple Health / Garmin / Zepp (часы)
- Shared knowledge_base.json (если есть)

Если у тебя `kb_status='private'` — биомаркеры и анализы в дашборде будут пустыми, потому что они хранятся **только у тебя на маке**. Это твой выбор приватности.

## Кто это видит

| | Видно |
|---|---|
| Ты — по своей ссылке `/mc/{твой_token}` | ✅ |
| Александр — admin-дашборд показывает ссылки всех юзеров | ✅ если family/early_user/owner |
| Случайный человек который угадает твой token | ✅ (поэтому если страшно — пересоздай) |
| Поисковики (Google) | ❌ страница помечена `noindex` |

## Когда лучше смотреть на Mini-app, когда на Dashboard

- **Mini-app** — для регистрации (что съел, что принял) и быстрого взгляда на день
- **Dashboard** — раз в неделю/месяц вдумчиво посмотреть динамику, перед визитом к врачу собрать картину
