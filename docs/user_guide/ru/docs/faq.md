# FAQ

## Это безопасно?

Botkin — приватный проект на личном сервере владельца. Доступ по приглашению, шифрование по HTTPS, токены Telegram. Третьим лицам данные не передаются.

## Где хранятся мои данные?

На сервере проекта в Германии (Hetzner). База данных PostgreSQL, ежедневные бэкапы. Фотографии еды и голосовые — на том же сервере, рядом с записями дневника.

## Кто видит мои данные?

- Вы.
- Владелец проекта — технический администратор сервера (Александр Лысковский). Может видеть данные при отладке, но не использует их.
- Никаких рекламных систем, аналитики, третьих лиц.

## Можно ли удалить аккаунт?

Да. В [mini-app](dashboard.md) → **Настройки** → **Удалить аккаунт**. Удаляются: профиль, все записи дневника, добавки, медиа, токены. Откатить нельзя.

Альтернатива — экспортировать всё перед удалением: там же кнопка **Экспорт данных** (JSON-архив).

## Платно ли это?

Нет. На текущем этапе Botkin бесплатен для приглашённых пользователей. Если проект вырастет в публичный сервис — об этом сообщим заранее в [журнале изменений](changelog.md).

## Есть ли open source?

Да: [github.com/Lyskovsky/Botkin](https://github.com/Lyskovsky/Botkin). Можно поднять собственную копию у себя на сервере. Документация для разработчиков — в репозитории, в папке `docs/`.

## А мобильное приложение?

Botkin не делает отдельное приложение — это сознательное решение. Telegram даёт чат, голосовые, фото, push-уведомления и mini-app — этого хватает. Меньше моста, меньше точек отказа.

## Что делать, если бот не отвечает?

1. Проверьте интернет.
2. Отправьте `/start` повторно.
3. Если не помогло — напишите владельцу проекта в Telegram: [@lyskovsky](https://t.me/lyskovsky).

## Я раньше пользовался @HealthVault_bot — что с моими данными?

Они на месте. 12 мая 2026 проект переименован из HealthVault в Botkin, бот переехал на [@Botkin_md_bot](https://t.me/Botkin_md_bot). База, история, настройки — те же. Старый бот больше не отвечает; перейдите на новый и продолжайте.
