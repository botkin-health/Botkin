# Mini-app Дневник

Mini-app — это полноценный интерфейс Botkin внутри Telegram: таблицы, графики, редактирование записей. Не требует установки отдельного приложения.

## Как открыть

В чате с ботом нажмите кнопку **Дневник** в меню (рядом со скрепкой). Откроется окно поверх Telegram.

Если кнопки нет — отправьте `/start` ещё раз, она появится в нижнем меню.

## Три вкладки

### Дневник

Лента приёмов пищи за день. По каждой записи видно:

- блюдо и порция;
- калории и БЖУ;
- время записи и способ ввода (фото / голос / текст).

Сверху — итоги дня и прогресс к норме. Свайп по записи — редактировать или удалить.

## Что важно

- Mini-app синхронизируется с ботом в реальном времени — записи из чата сразу видны в дневнике.
- Работает на iOS, Android и в Telegram Desktop.
