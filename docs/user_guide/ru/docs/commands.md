# Команды бота

Все команды вводятся в чате с [@Botkin_md_bot](https://t.me/Botkin_md_bot). Можно начать с `/` — Telegram сам подскажет.

| Команда | Что делает | Пример |
|---|---|---|
| `/start` | Запуск бота, приветствие, ссылка на mini-app. | `/start` |
| `/help` | Список команд и краткая справка. | `/help` |
| `/setup` | Первичная настройка: пол, возраст, рост, вес, цели. | `/setup` |
| `/profile` | Показать и отредактировать профиль. | `/profile` |
| `/day` | Сводка за сегодня: калории, БЖУ, активность. | `/day` |
| `/week` | Сводка за последние 7 дней с трендом. | `/week` |
| `/vitamins` | Журнал добавок: что и когда принято. | `/vitamins` |
| `/activity` | Шаги, тренировки, пульс — за день. | `/activity` |

## Записи без команды

Большая часть взаимодействия — без команд. Просто отправьте:

- **Фотографию еды** — попадёт в дневник питания.
- **Голосовое** — расшифруется и сохранится (еда, добавки, самочувствие — бот поймёт контекст).
- **Текст** — например, `выпил витамин D 5000 МЕ` или `съел два яблока`.

## Подсказки

- Чтобы поправить последнюю запись — ответьте на сообщение бота словами «нет, это было…» — он переспросит и обновит.
- Чтобы удалить запись — нажмите на неё в [mini-app Дневник](dashboard.md) и выберите «Удалить».
