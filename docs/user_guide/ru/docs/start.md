# Как начать

Botkin работает в Telegram. Всё, что нужно — пять минут и сам мессенджер.

## Шаг 1. Открыть бота

Перейдите по ссылке: [t.me/Botkin_md_bot](https://t.me/Botkin_md_bot) и нажмите **Start** (или отправьте `/start`).

!!! warning "Внимание: имя"
    Правильный бот — **@Botkin_md_bot**. Похожие имена без префикса принадлежат другим людям и не имеют отношения к проекту.

## Шаг 2. Короткая анкета

После `/start` бот спросит имя, возраст, рост и вес — это нужно для расчёта норм калорий и активности. Ответы можно поправить позже командой `/profile`.

## Шаг 3. Записать первый приём пищи

Три способа, выбирайте удобный:

=== "Фото"

    Сфотографируйте тарелку и отправьте боту. Botkin распознает блюда и предложит подтвердить.

=== "Голос"

    Запишите голосовое: «съел овсянку с бананом и кофе с молоком». Бот расшифрует и сохранит.

=== "Текст"

    Просто напишите: `омлет из 3 яиц, помидор, кофе`.

## Шаг 4. Посмотреть итоги

- `/day` — что было сегодня.
- `/week` — сводка за неделю.
- Кнопка **Дневник** под чатом — откроет [mini-app](dashboard.md) с таблицами и графиками.

Готово. Дальше — [список всех команд](commands.md).
