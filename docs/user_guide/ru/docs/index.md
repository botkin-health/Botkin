# Что такое Botkin

Botkin — это персональный трекер здоровья, питания и добавок, который живёт прямо в Telegram. Бот: [@Botkin_md_bot](https://t.me/Botkin_md_bot).

## Что умеет

- **Дневник питания.** Запишите приём пищи фотографией, голосом или текстом — Botkin распознает блюда, посчитает калории, белки/жиры/углеводы и сохранит в дневник.
- **Добавки и витамины.** Учёт регулярных приёмов: что, когда, сколько. Напоминания (по желанию).
- **Сводки за день и неделю.** Сколько вы съели, как двигались, как спали — одной командой.
- **Mini-app Дневник** прямо в Telegram: три вкладки — Дневник, Добавки, Настройки.
- **Интеграции:** Apple Health (через Health Auto Export), Garmin, давление, вес.
- **MCP-доступ** для Claude Desktop и Gemini Desktop — спрашивайте свои данные на естественном языке (в разработке, см. [страницу про MCP](mcp.md)).

## Для кого

Гайд написан для семьи, друзей и ранних пользователей. Botkin — приватный проект, не публичный сервис: данные хранятся на личном сервере владельца, доступ — по приглашению.
