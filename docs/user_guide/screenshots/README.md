## Где сохранить

`docs/user_guide/screenshots/{имя файла}.png`

Размеры:
- Mobile (бот / mini-app) — оригинал из iPhone (примерно 1170×2532)
- Desktop (дашборд / admin) — оригинал из браузера ≥1920×1080, можно crop'ать

## Соглашение

- Только PNG
- Без персональных данных других людей (только Александр и публичные UI)
- Размер файла <500 KB после оптимизации (используй `pngquant` или TinyPNG)
- При обновлении UI — пересохраняем скрин, имя файла то же
