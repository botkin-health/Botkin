# Анализы крови и медицинские обследования

Здесь хранятся результаты:
- Общего анализа крови (ОАК)
- Биохимического анализа крови
- Анализа на гормоны
- Анализа на витамины и микроэлементы
- Других медицинских обследований

## Формат файлов

Рекомендуется создавать файлы по датам: `YYYY-MM-DD-название-анализа.md`

Пример: `2025-01-05-общий-анализ-крови.md`
