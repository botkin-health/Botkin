# Лекарства и добавки

Здесь хранится информация о:
- Принимаемых лекарствах
- Витаминах и добавках
- Дозировках
- Графике приема
- Побочных эффектах
- Аллергических реакциях

## Формат файлов

Рекомендуется создавать файлы по названиям лекарств или по датам начала приема.

Примеры:
- `витамин-d.md`
- `омега-3.md`
- `текущие-лекарства.md`
