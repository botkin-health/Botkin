# Медицинские записи

Здесь хранятся:
- Диагнозы
- Назначения врачей
- Результаты обследований
- История болезней
- Выписки из больниц

## Формат файлов

Рекомендуется создавать файлы по датам или по врачам/специализациям.

Примеры:
- `2025-01-05-терапевт.md`
- `кардиолог.md`
- `выписка-2025-01-10.md`
