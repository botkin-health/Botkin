# 👤 A) ЛИЧНОЕ ЗДОРОВЬЕ И СОСТОЯНИЕ

### ❤️ Здоровье и Риски
- [ ] **Контроль Давления**:
    - [ ] Продолжать замеры утром/вечером.
    - [ ] [Ежедельно] Анализировать связь с "соленой едой" и алкоголем.
    - [ ] **Цель**: Стабилизировать < 125/80 (исключить пики >135).
- [ ] **Висцеральный жир**:
    - [ ] Следить за динамикой (цель: снижение с 14 до нормы <10).
    - [ ] Контроль корректности данных с весов (чтобы не путалось с Body Fat).

### 🥗 Питание
- [ ] **Эксперименты**:
    - [ ] [Гипотеза] Снижение соли в вечерних приемах пищи (тест -> замер давления).
    - [ ] [Гипотеза] Алкоголь: минимизация для улучшения качества глубокого сна.
- [ ] **База**:
    - [ ] Удерживать дефицит калорий и норму белка (>150г).
    - [ ] Контроль срывов: фиксировать триггеры переедания.

### 🏃 Спорт и Восстановление
- [ ] **Активность**:
    - [ ] Кардио/HIIT (подтверждено положительное влияние на давление).
    - [ ] **Восстановление**: Внедрить баню / массаж (1 раз в 1-2 недели).
- [ ] **Сон (Critical)**:
    - [ ] Спать > 7 часов (недосып дает +5 пунктов к давлению).
    - [ ] Режим: отбой до 00:00.

### 📊 Метрики
- [ ] **Регулярность**:
    - [ ] Вес: Ежедневно или через день (утром натощак).
    - [ ] Давление: Минимум 3-4 раза в неделю или при изменении самочувствия.
