### 1. Создать репозиторий на GitHub

1. Откройте: https://github.com/new
2. Название: **HealthVault**
3. Описание: `Personal health knowledge base`
4. Выберите: **🔒 Private**
5. **НЕ** ставьте галочки на README, .gitignore, license
6. Нажмите "Create repository"

### 2. Подключить к GitHub

После создания репозитория выполните:

```bash
cd /Users/alexlyskovsky/HealthVault
git remote add origin https://github.com/Lyskovsky/HealthVault.git
git push -u origin main
```

### 3. Начать заполнять данные

- Откройте `data/health/basic-info.md` и заполните основную информацию
