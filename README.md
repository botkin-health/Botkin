# HealthVault

Персональная база знаний о здоровье, спорте, анализах крови и образе жизни.

## Структура проекта

- `data/health/` - Общая информация о здоровье, симптомы, жалобы
- `data/blood-tests/` - Результаты анализов крови, медицинских обследований
- `data/sports/` - Тренировки, спортивные достижения, физическая активность
- `data/lifestyle/` - Образ жизни: питание, сон, привычки
- `data/medical-records/` - Медицинские записи, диагнозы, назначения врачей
- `data/medications/` - Информация о принимаемых лекарствах и добавках

## Использование

Это приватный репозиторий для хранения персональной медицинской информации.

## Безопасность

⚠️ **Важно**: Этот репозиторий содержит персональные медицинские данные. Убедитесь, что репозиторий приватный и доступ имеют только вы.
